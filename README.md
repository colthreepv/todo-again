Todo again
==========

Just gonna do Todo MVC all over again to evaluate possibile frameworks to use.

This one is: **React.js + Redux**

The repository includes a "base":
  * eslint
  * webpack
  * babel
  * jsx compilation
  * hot reloading for react
  * webpack-dev-server
  * react
  * app source code


To use it just:
  * `npm install`
  * `npm start`
  * open browser on `http://localhost:8080`
